#### 打开用户设置

+ ![win](./win.svg)： `ctrl+,`
+ ![apple](./apple.svg)： `⌘+,`

#### 打开最近的打开的项目

+ ![win](./win.svg)： `ctrl+R`
+ ![apple](./apple.svg)： `⌘+R`

#### 打开文件

+ ![win](./win.svg)： `ctrl+O`
+ ![apple](./apple.svg)： `⌘+O`


#### 打开新的Vscode
+ ![win](./win.svg)： `ctrl+shhift+N`
+ ![apple](./apple.svg)： ``

+ ![win](./win.svg)： `ctrl+K ctrl+O`
+ ![apple](./apple.svg)： `⌘+K ⌘+O`

+ ![win](./win.svg)： `alt+Z`
+ ![apple](./apple.svg)： `option+R`
