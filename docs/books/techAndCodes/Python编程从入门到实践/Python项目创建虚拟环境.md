1. 创建虚拟环境
 `python3.12 -m venv .venv`
2. 激活虚拟环境
 `source venv/bin/activate`
3. 选择虚拟环境作为解释器
+ 按 `CMD+Shift+P` 打开命令面板
+ 输入"Python: Select Interpreter"（VSCode 下方的菜单栏也可以选 Python 环境）
+ 选择刚刚创建的虚拟环境（通常带有"venv"标识）
