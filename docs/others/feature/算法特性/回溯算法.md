##### 回溯代码框架

```
void backtracking(参数) {
    if (终止条件) {
      存放结果;
      return;
    }
    for (选择本层集合中元素 (画成树，就是树节点孩子的大小)) {
      处理节点;
      backtracking();
      回溯，撤销处理结果;
    }
}
```
