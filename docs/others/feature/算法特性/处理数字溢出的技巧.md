一般是要判断是数字a是10倍的速度增长，判断    

`a > Integer.MAX_VALUE/10 || (a == Integer.MAX_VALUE/10 && a1 > 7)`
`a < Integer.MIN_VALUE/10 || (a == Integer.MIN_VALUE/10 && a1 < -8)`

以上条件成立，则说明a数字再增长10倍，就会溢出   
其中(a1是最后1位)     

::: info
